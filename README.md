# 🎮 Anime Memory Game

**Классическая игра на память с тематическими вселенными из аниме**

## 📖 О проекте

**Anime Memory Game** — это классическая игра на развитие памяти, где игрокам нужно находить пары одинаковых карточек. Игра поддерживает режимы для 1-4 игроков, различные размеры поля и три тематические вселенные из популярных аниме.

### 🎯 Цель игры
Открывайте карточки по две, запоминайте их расположение и находите все пары быстрее соперников!

## ✨ Особенности

### 🎮 Режимы игры
| Режим | Игроков | Описание |
|-------|---------|----------|
| **Соло** | 1 | Одиночная игра с отслеживанием общего времени прохождения |
| **1 vs 1** | 2 | Соревнование двух игроков, 30 секунд на ход |
| **1 vs 1 vs 1 vs 1** | 4 | Полноценная игра на четверых |

### 📏 Размеры поля

 2×2,  2×4,  4×4,  4×8,  8×8 

### 🎨 Тематические вселенные

| **One Piece** | 
| **JoJo** |
| **Mix** | 

### ⏱️ Система таймеров
- **Соло режим** — общее время прохождения всей игры
- **Многопользовательский режим** — 30 секунд на каждый ход
- Визуальное предупреждение при остатке 5 секунд (красный цвет)

### 🏆 Система очков
- +10 очков за каждую найденную пару
- Отслеживание счёта каждого игрока
- Автоматическое определение победителя

### Требования
- Windows 7 / 8 / 10 / 11
- .NET Framework 10
- Visual Studio 2026 (для разработки)
